Basic SAP Services

Our Managed Services support for SAP Basis

Coverage of SAP Basis Support : SAP Router, Solutions Manager, SAP S/4 Hana, Fiori & Web Dispatcher

SAP ECC & BW Administration:
  • Authorize user requests
  • Create user IDs
  • Modify user access
  • Reset passwords
  • Unlock IDs
SAP ECC & BW Transport Management:
  • Transport requests progress from DEV to UAT and finally to PROD
  • SAP Solution Manager ensures a systematic and well-documented process
  • Solution Manager acts as a collaborative hub, streamlining communication for consultants
SAP System Health Checks:
  • Daily monitoring and troubleshooting of ECC and BW systems as per checklist
  • Periodic EWA analysis and action items
  • SAP Performance Administration
SAP Marketplace Administration:
  • Raise SAP OSS messages
  • Administer access keys
  • Register developers
Printer maintenance in Production:
  • New Printer configuration
  • Troubleshooting issues occurring with existing printer
SAP HANA Management for BASIS Team:
  • Conduct daily health checks for HANA Appliance
  • Monitor overall DB system health, analyze errors/alerts, sessions, threads, and locks
  • Analyze log files for system start/stop, service errors, health checks
  • Monitor server resource utilization and memory consumption
  • Verify successful scheduling and running of backups
  • Manage efficient backup and restore using SAP Native tools
  • Implement snapshot-based backup and restore using professional tools
  • Provision disaster recovery in another region or AZ in another cloud account
System Updates & reporting:
  • Weekly Productions TR Logs
  • Weekly sensitive T-Code report to customer audit team
  • Weekly Status updates
  • Every Week sending the Produciton TR Logs
Other SAP Activities:
  • Perform quarterly client copy
  • Administer content server
  • Upgrade SAP Kernel
  • Implement S Notes/Plugins and SAP Security notes
  • Automate SAP system start & stop using cloud-native tools
